# Social Link Preview Metadata — Implementation Plan

## Summary

Add Open Graph and Twitter Card metadata to `index.html` and generate a 1200×630 social-preview image based on the actual Cover Page design. Production URL: `https://roja-projects-only.github.io/ePortfolio/`.

---

## Files to Create

### 1. `scripts/generate-og-image.mjs` (new)

A Node.js script that:
1. Starts the Vite dev server on a free port
2. Launches headless Chrome via Puppeteer
3. Sets viewport to 1200×630, emulates `prefers-reduced-motion: reduce` to skip animations
4. Navigates to `http://localhost:<port>/`
5. Waits for fonts and layout to settle (network idle + small delay)
6. Captures a full-page screenshot of the Cover Page section only (via element screenshot to avoid navbar chrome)
7. Saves output as `public/og-cover.png`
8. Shuts down the server and browser

**Why Puppeteer**: It captures the exact rendered output of the React app — same fonts (Source Serif 4, Inter), same Tailwind v4 tokens, same layout. No manual steps needed.

**Why `prefers-reduced-motion: reduce`**: The Cover Page uses staggered `motion` entrance animations. Emulating reduced motion makes all elements instantly visible, producing a clean static screenshot without timing fragility.

### 2. `public/og-cover.png` (generated)

- Format: PNG
- Dimensions: 1200 × 630 pixels
- Content: The actual rendered Cover Page — title "Beyond Limitations", subtitle, researcher names, metadata grid — styled identically to the live site on warm off-white canvas (#f7f6f2)

## Files to Modify

### 3. `index.html` — Add metadata in `<head>`

Complete metadata block to add (after existing `<title>` and `<meta>` tags):

```html
<!-- Open Graph -->
<meta property="og:type" content="website" />
<meta property="og:site_name" content="Beyond Limitations" />
<meta property="og:title" content="Beyond Limitations — A Philosophical Reflection on the Lived Experiences of Persons with Disability" />
<meta property="og:description" content="A student ePortfolio exploring phenomenology and the lived experiences of persons with disability through philosophical reflection on real interviews." />
<meta property="og:url" content="https://roja-projects-only.github.io/ePortfolio/" />
<meta property="og:image" content="https://roja-projects-only.github.io/ePortfolio/og-cover.png" />
<meta property="og:image:width" content="1200" />
<meta property="og:image:height" content="630" />
<meta property="og:image:alt" content="Beyond Limitations — A Philosophical Reflection on the Lived Experiences of Persons with Disability" />

<!-- Twitter Card -->
<meta name="twitter:card" content="summary_large_image" />
<meta name="twitter:title" content="Beyond Limitations — A Philosophical Reflection on the Lived Experiences of Persons with Disability" />
<meta name="twitter:description" content="A student ePortfolio exploring phenomenology and the lived experiences of persons with disability through philosophical reflection on real interviews." />
<meta name="twitter:image" content="https://roja-projects-only.github.io/ePortfolio/og-cover.png" />
<meta name="twitter:image:alt" content="Beyond Limitations — A Philosophical Reflection on the Lived Experiences of Persons with Disability" />

<!-- Canonical -->
<link rel="canonical" href="https://roja-projects-only.github.io/ePortfolio/" />
```

**Design decisions:**
- All image/URL references use absolute HTTPS production URLs (not relative paths) — required by social crawlers
- Description is concise, formal, student-appropriate, and does not expose participant identities
- `og:image:alt` matches the visual content of the Cover Page (title text)
- Twitter Card uses `summary_large_image` for the 1200×630 preview

### 4. `package.json` — Add devDependency and npm script

Add `puppeteer` as a devDependency:
```
npm install --save-dev puppeteer
```

Add a script entry:
```json
"generate-og-image": "node scripts/generate-og-image.mjs"
```

## Execution Order

1. Install Puppeteer: `npm install --save-dev puppeteer`
2. Create `scripts/generate-og-image.mjs`
3. Run `npm run generate-og-image` → produces `public/og-cover.png`
4. Edit `index.html` to add all OG/Twitter/canonical metadata
5. Run `npm run build` to produce the production build
6. Verify `dist/og-cover.png` exists (copied from `public/` by Vite)
7. Inspect `dist/index.html` — confirm all `<meta>` tags present with absolute URLs
8. Verify image dimensions: `1200×630`
